## Downloading

```bash
git clone https://github.com/Summer2023SHY/Automata.git
```

## Updating

```bash
git pull
```

## Running

```bash
mvn compile exec:java
```

## Testing

```bash
mvn test
```

## Creating and Opening Javadocs

```bash
mvn javadoc:javadoc
```

Then open `target/site/apidocs/index.html`

## Requirements

- Java 17 or later
- Apache Maven (if you are building from source)
